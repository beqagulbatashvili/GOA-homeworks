1. State
კომპონენტის შიგნით მოთავსებული ცვლადი, რომელიც იცვლება.

2. Setter function
ფუნქცია, რომელიც ამ state-ს ცვლის.

3. Hook
React-ის სპეციალური ფუნქცია, რომელიც დამატებით შესაძლებლობებს გვაძლევს.

4. Built-in hook
React-ში ჩაშენებული ჰუკი ( напр. useState, useEffect).

5. useState
ჰუკი, რომელიც ქმნის state-ს და მის შემცვლელ ფუნქციას.

6. Component re-render
როცა state იცვლება და კომპონენტი თავიდან იხატება.
